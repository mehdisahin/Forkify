## About The Project

![FireShot Capture 016 - forkify __ Search over 1,000,000 recipes - forkifyappp netlify app](https://user-images.githubusercontent.com/97400597/230788153-cf1720a3-9d06-4eef-b6b2-dba0d1314d38.png)
